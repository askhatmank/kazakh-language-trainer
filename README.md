# Тренажер казахского языка от DigitalRepetitor

Интерактивное веб-приложение для изучения базовой лексики казахского языка.

## Особенности
- Разделение на уровни (Основы, Семья, Числа).
- Случайное перемешивание вопросов и вариантов ответов.
- Индикатор прогресса прохождения уровня.
- Адаптивный дизайн (работает на мобильных устройствах).

## Как использовать
Просто откройте файл `index.html` в любом современном браузере.
